# User Registration and Login System Using PHP

The User Registration and Login System Using PHP is a versatile web application that serves as a solid foundation for implementing user authentication and management in your web projects. This system is designed to provide secure user registration, login, and user data management features, all powered by PHP and a MySQL database.

## Key Features

- User Registration: New users can sign up by providing their first name, last name, contact number, email, username, and password.

- User Authentication: Registered users can securely log in using their username and password.
- Password Hashing: Passwords are hashed using PHP's built-in password hashing functions to ensure security.
- Responsive UI: The system features a user-friendly, responsive user interface built with Bootstrap.
- User Listing: Registered users are displayed in a table, making it easy to manage user accounts.
- User Update and Deletion: Admins can update user information and delete user accounts when needed.
- Session Management: PHP sessions are used to maintain user login status throughout their interactions with the web application.
- Navigation: The application includes a simple navigation menu to switch between login and registration forms and to log out.


## How to Run?

- Download and Install XAMPP.
- Download the provided source code zip file.
- Note: The database is included on the provided zip file.

## Installation/Setup

- Transfer the provided source code to the htdocs folder.
- Create a database named "school_task_manager_db" in phpMyAdmin MySQL.
- Import the provided SQL file into the created database.
